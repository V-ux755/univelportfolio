CSS Specificity Project: Country Flags

In this project, I explored CSS specificity by applying gradient backgrounds to various sections, each representing the flag of a different country. Using pseudo-classes, I designed sections for Nigeria, Japan, Gambia, Gabon, France, Belgium, and Germany, ensuring that each section visually captures the distinct colors and patterns of the respective flags. This assignment highlights my understanding of CSS selectors and my ability to create engaging, themed designs.
